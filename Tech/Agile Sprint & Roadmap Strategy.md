# Agile Sprint & Roadmap Strategy: Ghorbari
**Workflow:** 2-Week Sprints bridging cross-functional AI Agents and human oversight.

## 1. Roadmap Phases
* **Phase 1 (MVP - Completion Phase):** Core platforms (Next.js Web & Flutter App), AI Image Generation implementation, Bilingual Support (Bengali/English), and Lead Magnet interactive tools (Construction & Interior Cost Calculators).
* **Phase 2 (Growth & Orchestration):** Robust Google Sheets Sales CRM integration, Workflow adjustments (e.g., Building Permit checkpoints), enhanced user authentication flows, and initial content automation (Content Calendar rollouts).
* **Phase 3 (Optimization):** Supply Chain tracking for site materials, advanced Analytics dashboards, and Deep AI predictive recommendations for renovation and construction paths.

## 2. Agent Instruction: "The Definition of Done"
A development task is not "Done" until:
1. The code logically executes without syntax errors.
2. The UI is tested for Responsive Design (Desktop, Tablet, Mobile) and strictly follows Dalankotha brand guidelines.
3. Relevant Supabase Database migrations/RLS policies are successfully applied.
4. The `task.md` or implementation roadmap artifact is updated to reflect progress.
