# UI/UX Design Philosophy: Ghorbari (Dalankotha)
**Core Aesthetic:** Geometric, Engineered, Transparent, and High-Affordance Workspace.

## 1. User Experience (UX) Rules
* **The "Offline-First" Site Reality:** Site engineers use the app in varying network conditions. Critical data input (e.g. tracking progress) must never be lost to a dropped connection.
* **Bilingual Inclusivity:** The entire UI must support fluid toggling and elegant typography wrapping for both Bengali and English, accommodating varying text lengths.
* **The 3-Click Rule:** Users should reach their core destination (Calculators, CRM, AI Consultant) in 3 clicks or less.
* **Error Handling:** Never show a "404" without a "Go Back" button. Use friendly, human language for technical errors.

## 2. Visual Identity (Dalankotha Brand)
* **Design System Integration:** Adhere to the locked Dalankotha brand aesthetics. This involves utilizing specific geometric typography and engineered layouts conveying authority.
* **Color Palette:** Utilize the exact HSL primary/secondary palette tailored for light and dark modes. Contrast must ensure readability in direct sunlight (for on-site workers).
* **Components:** Loading states must use Skeletons or Spinners; interfaces should employ subtle micro-animations to feel premium. Avoid generic placeholders.
